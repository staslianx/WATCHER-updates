Added weekly aggregation using Apple's fiscal calendar, expanded filtering and export capabilities, and improved UI responsiveness based on feedback.

Added

  • Weekly price aggregation using Apple fiscal week structure for trend analysis
  • Line of Business (LoB) filters for category-based product filtering
  • "Include Night Prices" toggle to control whether night-scraped sessions are included in calculations
  • "Consolidate Variants" toggle to merge color variants into single rows (uses most common price when variants differ)
  • Menu bar widget showing biggest price changes per product category
  • Notifications for database updates and configurable price change thresholds
  • "Collapse All" and "Clear Filter" buttons in product pickers
  • Keyboard shortcuts: arrow keys for picker navigation, CMD+O to open product URLs
  • Retailer IDs, PPN codes, quarter, and week numbers in Excel and CSV exports

Changed

  • Changes View now shows individual color variant price changes instead of aggregated changes
  • Historical price change exports moved from History View to Changes View
  • Simplified time period picker with fewer steps to change date ranges
  • iCloud Sync Monitor optimized for faster and more reliable sync on database updates
  • Bestseller Products filter now supports custom product selection
  • Settings interface reorganized for better navigation

Fixed

  • Missing base prices for iPhone 15 128GB and 13-inch MacBook Air 16GB 256GB variants
  • Market Events calendar showing only one indicator when multiple events exist on the same date
