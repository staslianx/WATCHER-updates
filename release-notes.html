Added

  • TouchID authentication for app security. WATCHER can now be locked with CMD+L and unlocked using TouchID biometric authentication. This optional feature provides an additional layer of privacy and must be enabled in Settings
  • Context menu (secondary click) in Snapshot View for quick access to product URLs and visualizations
  • Context menu for DG compliance period details by clicking the status icon
  • ALP selection and custom price configuration in DG Period settings

Changed

  • Average Discount chart redesigned with inverted vertical bars for clearer comparison
  • Simplified timeframe picker to W (Weekly) and Q (Quarterly) modes with dedicated week and quarter pickers for historical data navigation
  • Native color picker replaced with curated 18-color palette for a streamlined experience

Fixed

  • Market Events calendar not being visible when app window is not in fullscreen mode
  • Partner Filters not being applied when Product Filters are selected first
  • DG compliance being incorrectly applied to all retailers when only one retailer is selected
  • DG compliance being calculated even when the latest price record is Out of Stock or Marketplace
  • Visualization data gaps in weekly aggregation logic
  • Not being able to send feedback because of Linear authentication issue
